Como o arquivo JSON FOI INSTALADO (MAIOR ROLÊ MISERICORDIA)

1- NODE.js: Foi baixado no computador, pois sem ele não é possivel utilizar SEM o npm 

2- AUTORIZAR POWERSHELL COMO ADMINISTRADOR
    2.1 - apos autorizar, execute o comando:
    Set-ExecutionPolicy RemoteSigned

    2.2 escreva A e aperte ENTER

3- Vá para o terminal do vs code
    3.1 - escreva npm init
    3.2 - escreva npm i -g json-server

4- vá no seu arquivo json e escreva
    4.1 - json-server --watch api-artists/artists.json --port 3000
